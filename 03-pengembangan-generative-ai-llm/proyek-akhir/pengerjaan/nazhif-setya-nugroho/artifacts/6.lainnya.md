Submission yang Tidak Sesuai Kriteria
Jika tidak sesuai dengan kriteria, submission Anda akan ditolak oleh reviewer. Berikut poin-poinnya.
1. Tidak melampirkan file yang diminta pada ketentuan berkas submission.
2. Terdeteksi menggunakan platform No-Code/Low-Code, AutoML (seperti Hugging Face AutoTrain), atau UI tools instan lainnya untuk melakukan fine-tuning.
3. Dokumen notebook tidak dijalankan terlebih dahulu, sehingga output sel kode tidak terekam.
4. Tidak mengimplementasikan fitur wajib sesuai level, mulai dari Basic, lalu Skilled, hingga ke Advanced.
5. Menggunakan file dokumen lain atau hanya menggunakan beberapa dokumen yang disediakan sebagai knowledge RAG.
6. Menggunakan dataset selain yang telah disediakan untuk melakukan fine-tuning dan reinforcement learning GRPO.
7. Terdeteksi menggunakan model instruct/chat hasil fine-tuning pihak ketiga, atau siswa lain di Hugging Face yang digunakan untuk submission ini.
8. Tautan (link) model hasil fine-tuning yang diunggah ke Hugging Face bersifat Private, salah ketik, atau tidak disertakan.
9. Tidak menyembunyikan atau tidak menggunakan environment variables untuk menyimpan API Key seperti Hugging Face Token.
